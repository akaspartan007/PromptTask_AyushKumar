
---

## Local Deployment

Run:

```bash
python golden_response.py
```

Then open generated HTML files locally or host them using:

- GitHub Pages
- Netlify
- Static web hosting platforms

---

## Deployment Pipeline

```text
Local Repository
        ↓
Golden Benchmark Generation
        ↓
Static Frontend Build
        ↓
Netlify / GitHub Pages Deployment
        ↓
https://goldenresponsetraining.netlify.app
```

No backend server, database server, or build tooling is required.
